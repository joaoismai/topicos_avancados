# **Tópicos Avançados de Computação**
## FLOW INDEX

O Flow Index é um indicador proprietário desenvolvido para este projeto que quantifica a qualidade ambiental de um espaço de trabalho e o seu impacto direto na produtividade e bem-estar dos colaboradores.

Em vez de olhar para dados isolados (como apenas a temperatura ou apenas o ruído), o Flow Index utiliza algoritmos de processamento de dados IoT para fundir múltiplos sinais sensoriais numa métrica única e intuitiva de 0 a 100.

### Como é calculado?
O índice baseia-se em quatro pilares fundamentais monitorizados em tempo real:

Conforto Térmico (Fator Humano): Utiliza os modelos PMV (Predicted Mean Vote) e PPD (Predicted Percentage Dissatisfied) para perceber se a temperatura e humidade são ideais para o corpo humano.

Qualidade do Ar (Fator Cognitivo): Monitoriza os níveis de CO₂ e compostos orgânicos. Níveis elevados de CO₂ estão diretamente ligados à sonolência, perda de concentração e erros de decisão.

Ambiente Acústico (Fator de Foco): Mede a pressão sonora média (dB) para detetar poluição sonora que interrompa o estado de "fluxo" (Deep Work) da equipa.

Luminosidade (Fator Ergonómico): Monitoriza os níveis de Lux para garantir que a iluminação é adequada à tarefa realizada, evitando a fadiga ocular e aumentando o alerta natural do utilizador.
